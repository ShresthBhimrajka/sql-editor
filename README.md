A react project for an online sql editor

## Dependencies

  - react-ace
  - react-hot-toast
  - react-csv
  - feather-icons-react

## Page Load

  - Page load time - 858ms 
  - Calculated using network panel in google chrome
  - Used a minimalist design to avoid excess components on page load
